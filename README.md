# Plugin-ACCV-Controle-de-Chat---MINECRAFT-
Plugin ACCV foi criado para ajudar servidores a controlar acesso ao chat de seus servidores perante nova Lei 15.211/2025, onde deveram restringir chat para menores de 18 anos. Este plugin não apenas cria uma white list de chat mas cria varios canais com pativeis com as idades e sistema de moderação complexa.
